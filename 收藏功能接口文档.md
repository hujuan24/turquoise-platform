# 收藏功能接口文档

## 1. 接口概述

本接口文档描述了商品收藏功能的后端API，包括添加收藏、取消收藏、查询收藏状态等操作。

## 2. 接口列表

| 接口名称 | URL | 请求方法 | 功能描述 |
| :--- | :--- | :--- | :--- |
| 添加收藏 | /collect/add | POST | 添加商品到收藏列表 |
| 取消收藏 | /collect/remove | POST | 从收藏列表中移除商品 |
| 检查收藏状态 | /collect/check | GET | 检查用户是否已收藏指定商品 |

## 3. 接口详情

### 3.1 添加收藏

**URL**: `/collect/add`

**请求方法**: POST

**请求参数**:

| 参数名 | 类型 | 必需 | 描述 |
| :--- | :--- | :--- | :--- |
| productId | Long | 是 | 商品ID |

**请求示例**:

```http
POST /collect/add
Content-Type: application/x-www-form-urlencoded

productId=10912
```

**响应格式**:

```json
{
  "resultCode": 200,
  "message": "SUCCESS",
  "data": null
}
```

**错误响应**:

```json
{
  "resultCode": 500,
  "message": "用户未登录",
  "data": null
}
```

## 5. 前端调用示例

```javascript
// 添加收藏
function addFavorite(goodsId) {
    $.ajax({
        type: 'POST',
        url: '/collect/add',
        data: { productId: goodsId },
        contentType: 'application/x-www-form-urlencoded',
        success: function (result) {
            if (result.resultCode === 200) {
                Swal.fire({
                    text: "收藏成功",
                    icon: "success",
                    iconColor: "#1d953f",
                });
                document.getElementById('favoriteBtn').classList.add('active');
                document.getElementById('favoriteBtn').textContent = '❤️ 已收藏';
            } else if (result.message === '用户未登录') {
                Swal.fire({
                    text: "请先登录",
                    icon: "info",
                    iconColor: "#3b82f6",
                    showCancelButton: true,
                    confirmButtonText: '去登录',
                    cancelButtonText: '取消'
                }).then((flag) => {
                    if (flag.value) {
                        window.location.href = '/login';
                    }
                });
            } else {
                Swal.fire({
                    text: result.message || "收藏失败",
                    icon: "error",
                    iconColor: "#f05b72",
                });
            }
        },
        error: function (xhr, status, error) {
            Swal.fire({
                text: "操作失败，请稍后重试",
                icon: "error",
                iconColor: "#f05b72",
            });
        }
    });
}
```
